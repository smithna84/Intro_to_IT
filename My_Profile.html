Project Idea

Overview

This project is to develop and building a navigation application linked to earbuds with built-in cameras for the visually impaired. The application will be able to guide the user to a location with voice directions but also advise how to avoid an object as well as assistance with boarding trains and buses. The application will also be able map new locations such as houses, apartments and public buildings. Application will be voice activated and configured to accept only the hosts voice.

Motivation

The motivation behind this project is to develop artificial intelligence that will benefit the visually impaired. It is estimated that there is over 575,000 people who are blind or visually impaired. While there has been massive in roads in technology for the blind, I feel there is still a long way to go in bridging the gap of what is currently available and what is possible that will allow the blind to navigate this world as a non-visually impaired person can. The day where you cannot tell someone walking next to you is blind is something we should be striving for.

Description

This project will develop a mobile application that will link to earbuds that have multiple cameras in each earbud. The application will use the phones built-in navigation to determine location and guide the user to their destination. With the multiple cameras in the earbuds, the application will be able to determine what is an object in front of them that the user needs to navigate around but also what is to their left and right as well as behind them. If public transport is required, the user will be guided to where the train doors or bus door is, and which is an available seat. The application will be able to map out the floor plan of houses, apartments and public buildings. This will allow for the user to be guided to bathrooms when visiting friends, and other rooms. The floor plans will be saved for future use. The application will be voice activated and mapped to the users’ voice; this will prevent the application from picking up any other voices when in crowded places. As the application learns, the user’s regular appointments and destinations will be stored and be able to automatically trigger based on the devices date and time and direct the user. Calendar integration will allow for automatic navigation for appointments with a location.

Tools and Technologies

Application will be built from scratch.

Skills Required

For this project, proficient skills in programming will be required to create the basis of the application. There will also be efficiency in machine learning algorithms required to develop means for the application to learn what an object is, how to avoid objects and. Experience in engineering design as well as manufacturing the earbuds will also be needed.

Outcome

When completed, further independence will be available to the visually impaired with users of the application being able to navigate the world without the assistance of others. The concepts of the application and multiple cameras could open the door to expanding the application to allowing a visually impaired person to drive. All though this idea would make the Guide Dogs associations’ work in training dogs to see for the visually impaired redundant, it could open the path for the association to change direction and work with them in developing the application further with the hardware only available for sale through the Guide Dogs Association.
